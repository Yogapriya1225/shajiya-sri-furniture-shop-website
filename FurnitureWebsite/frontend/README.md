# Shajiya Sri Furniture Mart — Frontend

React 19 + TypeScript + Vite + Tailwind CSS + shadcn/ui frontend for the
Shajiya Sri Furniture Mart showroom website. This is a showcase site —
there is no cart, checkout, or online payment; every product links out to
a WhatsApp enquiry or a phone call.

See the root [`README.md`](../README.md) for full setup and deployment
instructions covering both the frontend and backend together.

## Quick start

```bash
npm install
cp .env.example .env      # then edit VITE_API_BASE_URL
npm run dev
```

## Scripts

| Command           | Purpose                          |
|-------------------|-----------------------------------|
| `npm run dev`     | Start the Vite dev server         |
| `npm run build`   | Type-check and build for production |
| `npm run preview` | Preview the production build locally |
| `npm run lint`    | Run Oxlint                        |
